## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn
- Adobe Commerce account and API credentials

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file with your Adobe Commerce credentials
4. Start the development server:
   ```bash
   npm run dev
   ```
